定时器Timer

发表于 2018-04-07 | 更新于: 2018-04-07 | 分类于 Qt

###定时器用于实现定时操作，如：

  • 每三分钟保存一次
  • 每500毫秒闪烁一次
  • 每1秒刷新一次界面

    新建Timer

(1) 重写timeEvent()函数
virtual void timerEvent ( QTimerEvent * event );

(2) 启动定时器，指定时间间隔
m_timerId=startTime(500);

例子：实现一个文字时钟，显示系统当前时间
